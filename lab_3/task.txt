Реализация информационной базы

Структура файлов:
   1. FIO.DAT - символьные строки (фамилии студентов), разделенные `\n`
   2. GROUP.DAT - символьные строки (номера групп), разделенные `\n`
   3. LINK.IDX - записи структуры:

struct Link {
   unsigned long name_idx;  // индекс записи в FIO.DAT
   unsigned long group_idx; // индекс записи в GROUP.DAT
};

Функциональность программы:
   1. Включение новой фамилии студента с указанием группы
      - Добавление фамилии в файл FIO.DAT
      - Создание соответствующей записи в LINK.IDX
      - Связывание с существующей группой
   2. Добавление новой группы
      - Добавление номера группы в файл GROUP.DAT
      - Проверка на уникальность номера группы
   3. Поиск студентов по группе
      - По заданному номеру группы вывод всех фамилий студентов
      - Поиск через индексный файл LINK.IDX

Принцип работы:
   - Индексы представляют собой позиции в файлах, где начинаются соответствующие записи
   - Записи в FIO.DAT и LINK.IDX имеют взаимно-однозначное соответствие
   - Для поиска студентов группы осуществляется:
      1. Поиск индекса группы в GROUP.DAT
      2. Поиск всех записей в LINK.IDX с данным group_idx
      3. Извлечение фамилий из FIO.DAT по найденным индексам name_idx
