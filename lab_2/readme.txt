Задание: 

Написать программу, которая для заданного текстового файла, имя которого
вводится с клавиатуры, строит в оперативной памяти массив записей (структур)
вида <номер_строки, количество_символов_имеющих_повторы>, анализируя
строки файла. Отсортировать полученный массив по возрастанию значений поля
"количество_символов_имеющих_повторы", результирующий массив вывести на
экран. Для сортировки элементов массива использовать стандартную функцию в
стиле языка С qsort().
