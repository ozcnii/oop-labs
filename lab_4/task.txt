Написать программу управления информацией, состоящей из трёх списков:
    1. Динамически формируемый список спортсменов с записями вида:
        {<Фамилия И.О.>;}
    2. Динамически формируемый список учебных групп с записями вида:
        {<Вид_спорта>;}
    3. Динамически формируемый список индексных записей вида:
        {<индекс_спортсмена>; - порядковый номер записи в списке спортсменов
        <индекс_вида_спорта>; - порядковый номер записи в списке видов спорта}.

Программа должна обеспечивать выполнение следующих функций:
    1. Добавление нового спортсмена с указанием вида спорта.
    2. Изменение спортсмену вида спорта.
    3. Вывод на экран фамилий спортсменов по заданному виду спорта.
    4. Вывод на экран упорядоченного по алфавиту списка всех спортсменов с указанием их видов спорта.
    5. Вывод на экран упорядоченного по алфавиту списка спортсменов, занимающихся указанным видом спорта.

Для сортировки написать собственную функцию qsort.
